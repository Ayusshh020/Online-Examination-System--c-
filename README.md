# Online Examination System (C++)

## 📌 Project Overview

This project is a **console-based Online Examination System developed using C++**.
The system simulates a basic digital examination platform where students can answer multiple-choice questions and receive their score instantly.

The main goal of this project is to demonstrate the practical implementation of **core C++ programming concepts such as arrays, loops, conditional statements, and functions** while automating the examination process.

---

## 🎯 Objectives

* To create a simple and efficient **online examination system**
* To implement **MCQ-based questions**
* To automate **answer validation and score calculation**
* To demonstrate the use of **arrays, loops, and functions in C++**
* To improve understanding of **basic program flow and logic building**

---

## ⚙️ Features

* Console-based user interface
* Multiple-choice questions (MCQ format)
* Automatic answer checking
* Instant score calculation
* Input validation for user responses
* Simple and user-friendly program flow

---

## 🧠 Concepts Used

This project demonstrates the implementation of the following C++ concepts:

* Arrays for storing questions and answers
* Loops for displaying multiple questions
* Conditional statements (if-else) for answer validation
* Functions to organize the program structure
* Input/output handling using standard C++ libraries

---

## 🔄 Program Flow

1. The program starts from the `main()` function.
2. The user is prompted to enter their name.
3. The exam begins with a set of multiple-choice questions.
4. The user selects answers for each question.
5. The program checks the answers automatically.
6. The score is calculated based on correct responses.
7. The final result is displayed to the user.

---

## 🗂️ Code Structure

The project uses the following main components:

* **main() function**
  Controls the overall execution of the program.

* **startExam() function**
  Handles the examination process including displaying questions and calculating the score.

* **clearInput() function**
  Used to manage input errors and clear invalid inputs.

* **Arrays**
  Store questions, options, and correct answers.
